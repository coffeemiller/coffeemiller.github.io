## 게스트스피치 / 이동산 리더 (1/11)

### 암웨이의 가치
+ "암웨이 사업은 나이를 먹어야 가치를 느낀다"
  - 30 / 40 / 50 의 나이에 겪게되는 일들
  - 안타까운 주변의 현실들
  - 오래사는 것도 중요하지만, 어떤 삶을 사는 것인지가 중요하다

+ 남자들의 소원
  - 자고 싶을 때 자고, 깨고 싶을 때 깨는 삶
  - 지금 나는 내가 하고 싶은 일을 하면서 내가 자고 싶고 깨고 싶은 시간을 산다

+ 이별을 준비
  - 가족의 이별을 시간적 여유로 함께 할 수 있다는 감사
  - 충분히 슬퍼하고 함께 할 수 있다는 시간이 있다는 것은 감사할 일이다
  - 암웨이가 주는 보상


### 나에 대해서
+ 전북 김제 출신
  - 반복구매의 '성냥공장' 운영의 아머지
  - '라이터' 출시로 사업의 실패
  - '나무젓가락' 사업으로 변환 했지만, '쇠젓가락'으로 두번 실패

+ 집안의 책임
  - 어머니가 일을 하시게 됨
  - 부모님에 대한 측은함이 커지게 됨
  - 열심히 공부해서 '부모님에 대한 효도'를 결심
  - 행정고시, PD를 준비

+ 하나은행
  - IMF의 시작
  - 보증과 빚에 대한 압박
  - 부모님을 향한 그림들이 산산조각
  - 회사와 가정이 불안한 상태
  - 주식으로 인한 실패


#### 암웨이 만남
+ 우연한 만남
  - 불안한 상황 속에서 암웨이를 만남
  - 암웨이 사업설명을 준비 못한채 듣게 됨
  - 처음 본 '암웨이'에 대해 있는 그대로 받아들임
  - '부업'으로 할 수 있는 일에 대한 관심
  - 나에겐 무조건 해야 하는 일이기 때문에 열심 듣게 됨

+ 나에게 암웨이
  - 1억을 벌 수 있다는 희망
  - 미래 와이프는 반대
  - 다음 미팅에 와이프와 듣고 제대로 이해하게 됨

+ 미팅
  - 암웨이 사업에서 빠뜨려서는 안될 것
  - 아무리 내가 사업설명을 잘해도, 미팅이 없으면 확실하게 믿음을 쌓아줄 수 없다
  - 우여곡절 끝에 사업을 시작

+ 거절
  - 친구들과 주변인들에게 소개
  - 해도 해도 안되는 느낌의 거절
  - 스폰서 : 가족부터 시작하라는 제안
  - 어려운 용기를 내게 됨

+ 가족사업
  - 우연하게 갖게된 가족 모임(고스톱)
  - 간절함이 있기에 가족을 만날 자신감
  - 전지에 사업설명의 시작
  - 예상 외로 긍정적인 '가족'
  - 작은 누나를 통해 레그발견

+ 대전에서의 사업
  - 완벽성 보다 '열정'과 '진정성'에 감동한다
  - 누나 그룹의 15% 성장

+ 사업자 = 소비
  - 잘 되어가는 니 스스로 자만하게 됨
  - 파트너들이 하나 둘씩 떠나기 시작함
  - 초기사업 보다 더 힘든 시기
  - 이때 사람들은 많이 포기한다
  - 하지만 나에게 이것 아니면 없었다.


### 나에게 암웨이 = 나에게 모든 것
+ 구제
  - 암웨이 사업은 다이아가 된다고 구제되는 것이 아니다
  - 결심하는 순간 구제된다

+ 힘들었지만 지켜야하는 것
  - 미팅

+ 암웨이 여행
  - 자신감이 커짐
  - 멀어졌던 파트너을 다시 만날 용기

+ 다시 시작
  - 자신의 부족함을 알게 됨
  - 사라진 파트너들에게 미안함
  - 원망하지 않고 다시 연락

+ 긍정에 비중을 두자
  - 만나기 싫음에 비중을 두지 말고, 조금의 긍정에 초점
  - 반대했던 파트너가 이제는 성공하는 파트너로

+ 은행 후배
  - 긍정적인 미팅
  - 주변의 강한 반대
  - 5년이 지나 연락처를 알아내어 연락하게 됨
  - 출산 후 백일이 지나 만남
  - 다시 사업설명을 보여주고 변화된 암웨이에 대한 것을 제시
  - 변한 것 = 우리

+ 시험
  - 비참한 상황을 겪고 두려움이 사라짐
  - 역경을 대하고 해석하는지가 중요하다
  - 알고 있는 사람한테 문전박대는 나를 강하게 만든다
  - 훨씬 자신감있게 사업을 하게 됨

+ 암웨이 사업은 결국은 통하게 되어 있다

### 암웨이 사업
+ 어떻게 하면 잘 할 수 있을까?

+ 책임감
  - 암웨이를 선택한다고 누구나 다 성공할 수 없다
  - 10Core 습관을 다 하고 있다
  - 그러나 누구난 열심히하고 누구는 멀리한다
  - 성실한 습관으로 책임감이 필요하다
  - 책임감을 갖고 있는 사람은 쉽게 포기하지 않는다
  - 부정적인 해석을 멀리 한다
  - 어떻게든 긍정적인 결과를 내야하는 당사자
  - "성공하는 사람들의 7가지 습관" -스티븐 커비
    - '자녀 교육의 핵심적인 목표 2가지'
      1. 자립심
      2. 책임감
  - 성인들이 가장 중요하게 가져야할 덕목이 '책임감'이다
  - 암웨이에서 중요한 것은 '자립심'과 '책임감'이다.

+ 꿈
  - 꿈이 있는 것도 중요하다
  - 꿈에 대한 책임감이 강해야 한다

+ 목표
  - 누구나 목표는 다 있다
  - 목표에 대한 책임감이 있다면 성공이 달라진다

+ 꿈에 대한 목표를 설정하고, 그 목표를 위한 '책임감'을 통해 성공 할 수 있다.

+ 뱁새 VS 황새
  - 당신은 봽새인가 황새인가?
  - 우리는 황새이다.
  - 내가 어떤 마음가짐으로 생각하느냐에 따라 다르다
  - 내 스스로가 어떤 자세로 어떤 책임감으로 임하느냐에 달려있다
  - 다이아몬드 이상의 목표!

+ 책임감이 강한 사람은 결과에 탓을 하지 않는다
  - 모든게 내 책임
  - 결과에 대해 긍정적인 선택을 함

+ 내 안에 늑대 두 마리
  - 폄하, 자기비하, 비판적인 늑대
  - 긍정적인, 발전적 늑대
  - "누가 이길까?" -> "밥주는 늑대"
  - 우리 안에는 두 가지가 다 존재하다.
  - 문제는 어디에 기울이는지가 중요하다
  - 긍정적인 해석과 노력이... 긍정적인 결과를 얻을 떄 까지 포기하지 않도록 만든다

+ 내가 어떻게 해석하느냐에 달려있다
  - 활용여부에 달려있다


### 책임감은 문제에 대한 자세를 결정한다
+ 문제는 있다
  - 문제를 겪어 왔기 때문에 지금 이 자리에 있다.
  - 성공의 반대는 실패가 아니라 '포기'다
  - 부자의 반대는 빈자가 아니라 '도망자'이다
  - 문제에 대해서 회피하고 도망하는자가 빈자
  - 성장하기 위해서는 문제를 받아들이고 해결하는 능력을 키워라

+ "어떻게 할 수 있을까?"
  - 긍정적으로 방법을 찾아라
  - "먼저 할 수 있다고 믿어라. 그런 후에 방법을 찾아라" -링컨

+ 인생은 딱 한 번
  - 선택의 여지가 없다
  - 돈과 자유의 시간을 기반
  - '잘 살아야 한다'는 책임
  - '잘 살아야 한다'는 믿음을 가져라
