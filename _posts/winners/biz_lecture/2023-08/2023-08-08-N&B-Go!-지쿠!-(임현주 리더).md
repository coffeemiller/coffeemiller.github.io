+ Getting Stared! 뇌 오작동 알아차리기
  - 꿈! 사업확신! 포기와 유보(댓가지불)!
  - 도전, 실패 = 두려움, 죽음
  - 평판 잃음 = 소외감과 생존불리
  - 도파민 = 산만한 것이 생존유리
  - 과도한 식욕 = 생존유리
  - 손실회피편향 = 손해볼까봐 행동못함

+ 직장인, 자영업 -> 사업가로 이동의 진실
  - 처음부터 직상승하지 않고 떨어지고 난 후에 날아오름
  - 어린새가 하늘을 나는 원리
  - 우상향의 기대치에 매몰되지 마라

+ 제품애용, 꿈의 가시화, 시간확보
  - 우리가 모델이 되는 사업
  - 내가 제품을 애용해야 모델이 될 수 있다
  - 나에게 꿈을 이루는데 불필요한 것들은 버려라
  - 시간확보 : 시간을 허비하는 것들을 버리고 꿈에 집중

+ Dream Building(꿈 가꾸기)방법
  - 구체적인 꿈 : 갖고 싶은 것, 가고 싶은 곳, 하고 싶은 것, 배우고 싶은 것, 되고 싶은 모습
  - 간절한 꿈 : 안되면 너무 속상할 것 같은 상황, 이것만은 꼭 되야되는 일, 대내적인 꿈
  - 커다란 꿈
    - Small Dream : Big Problem
    - Big Dream : Small Problem
    - Super Dream : No Problem

+ Getting Started (사업 시작하기) 사업초기에 유의할 사항
  - 미팅
    - 사업은 단거리가 아닌 장거리 마라톤
  - 방향
    - 사막을 통과할 때는 오아시스의 중간 목표로!
  - 상담
    - 이겨내기 위해 스폰서를 통해 지혜얻기

+ Goal Setting(목표 세우기)
  - <초기사업자를 위한 목표 설정>
    - 자신의 사업 Tool 구축 및 미팅 참석 등 시스템 조인 목표 세우기
    - 3개월 이내 12% 달성 : 전제품 애용과 애용자 확보
    - 홈미팅 계획 : 스폰서 파트너의 도움으로 가능한 빨리 홈미팅 하기
    - 최소 월간 10회 이상의 사업설명과 사업설명미팅 초대
  - <SP루비의 목표설정>
    - SP 달성 날짜를 미리 정하고 1000만PV를 달성하기 위한 회원숙자와 사업설명 횟수를 중간 목표로설정
      - --> 150만PV 브론즈 파운데이션 ->* 7조 = 1050만 PV SP!
    - 안정된 F.PT네트워크는 소비자와 사업자의 균형
    - F.PT는 2000만PV의 루비를 목표 : FDiamond의 초석
  - 된다고 믿고 실천하는 것이 중요함

+ 프로슈머(9%)가 되는 과정 : 축적
  - 내가 10만 PV 소비
  - 나처림 10만 PV소비하는 소비자 15명 구축
  - 소비자들의 특징
    - 가입안하고 쓴다.
    - 신뢰 쌓고 관계 맺고 자주 보는 사이가 된다
    - 니즈 피악과 인터뷰한다
    - 경우에 따른 사업설명을 한다.

+ 프로슈머(9%) 복제되는 과정
  - 복제 특징
    - A,B : 현명한 소비를 목표로 프로슈머학습
    - 나 : 복제를 일으키는 사업자
    - A,B : 학습단계에서 배울 내용(제품, 사업설명, 간단한 요리, 데몬, 이의답변 등등)
    - 나 : A,B의 명단과 홈미팅하고 사업을 진행한다. 즉 A,B가 직접홈미팅을 아직 하지 못한다. 사업자가 아니다.
    - 필요한 것을 쓰는 소비자로 가득한 네트워크를 만든다. 그중에 애용자들에게 소비지출을 수익으로 다시 찾아오는 프로슈머 비젼을 전달.

+ 1000만 네트워크 그림 완성!
  - SP 네트워크 : 150만점 프로슈머 네트워크 7명 복제
  - 9프로 프로슈머 : 시스템에서 배우고 복제할 행동을 하면 사업자가 된다.

+ Goal Setting(목표 세우기)
  - <F.Diamond 핀을 목표로>
    - 핀 달성 시한을 정하라

    - 핀을 달성하기 위해서 어느 정도의 행동이 필요한지 알아야 한다. 하루 중 제한 된 시간만 사업에 투자하면서 단기간에 파운더스 다이아몬드 핀을 성취하겠다는 것은 무리한 목표

+ 누구를 후원 할 것인가?
  - 내 시간과 자원을 옮기는 것 : 후원

    - 열린 마음으로 가회에 관심을 두는 경향이 강하며 꿈이 크고 자신을 찾아온 사람을 위해서라도 기꺼이 시간을 내는 사람
